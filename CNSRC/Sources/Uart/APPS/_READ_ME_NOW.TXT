All example source code is the APPS directory. 

Project files for Microsoft Visual C/C++ (version 6.0) and Microsoft VC.Net can be found in the APPS directory.

Makefiles or project files for other compilers can also be found in the following ZIP files:

MICROSOFT60.ZIP : Command line makefiles for Microsoft Visual C/C++ versions 4.0 through 6.0
  BORLAND50.ZIP : Command line makefiles for Borland C/C++ version 5.0
  BORLAND55.ZIP : Command line makefiles for Borland C/C++ version 5.5
  C-BUILDER.ZIP : Project files for Borland C++ Builder.
  LCC-WIN32.ZIP : Command batch files LCC-WIN32
   WATCOM11.ZIP : Command line makefiles for Watcom C/C++ 11.0
   
See EXAMPLES.TXT for a brief list of example programs. Also refer to the Programmer's Manual (WSC_4C) found in the \DOCS directory for more details on each example program.

Also refer to section 3.0 of the WSC4C Programmer's Manual (WSC_4C.DOC or WSC_4C.HTM) for help in compiling the example programs.
