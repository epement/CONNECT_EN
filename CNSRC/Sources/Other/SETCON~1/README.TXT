INTRODUCTION
============
The SetConsoleTitle Api can be used in a Console 
application (Project Build settings, Link tab, Character 
radio button) to change what is displayed in the 
application Window Title Bar.

SOURCE FILES:
=============

Program Files	Description
=============	===================
Ctitle.cbl      Application program

REQUIREMENTS:
=============
Net Express  3.1

OPERATION:
==========
Animate the program and you will see the Application Outout 
title bar updated.
