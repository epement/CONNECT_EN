Local associations, используется горячая клавиша Shift+F9.

3. Вызов User Screen правой кнопкой мыши на левой/правой рамке окна.

4. Одиночное и групповое переименование файлов в файлы без имени вида
.extension.

5. Добавлена опция для сплита: вместо размера можно указывать число частей:
/#5 (на 5 частей).

6. Options - File Manager - Keys
[X] CapsLock+character activates incremental search.

7. Options - Window settings
[X] New-style icons and number
определяет способ отображения и поведение иконок на верхней части рамки
немодальных окон программы при нажатии на них кнопками "мыши":

 ╔[■1]════[]╗

Для окна File Manager нажатие правой кнопкой "мыши" на символы
иконок рамки выполняет следующие действия:

 [■ - переход левой панели в корневой каталог текущего диска;
 1] - переход левой панели в каталог на один уровень выше;
 [ - переход правой панели в корневой каталог текущего диска;
 ] - переход правой панели в каталог на один уровень выше.

Для остальных окон нажатие правой кнопкой "мыши" на символы
иконок рамки выполняет следующие действия:

 [■ - эмуляция нажатия на клавиши Ctrl+PgUp;
 1] - эмуляция нажатия на клавишу PgUp;
 [ - эмуляция нажатия на клавиши Ctrl+PgDn;
 ] - эмуляция нажатия на клавишу PgDn.

Нажатие левой кнопкой "мыши" на символ номера окна эмулирует команду
Previous (переход к предыдущему окну). Если при этом нажать и удерживать
клавишу Shift, будет выполнена команда Next (переход к следующему окну).

8. При нажатых Alt+Ctrl+Shift и режиме панели Descriptions
во всплывающем хинте показывается Description.

9. Panel sort and display options
() Name   [X] Dirs always - всегда сортировать каталоги только по имени,
независимо от выбранного режима сортировки;
() Digital number [ ] As string - рассматривать номер в имени как строку
(file2 > file10).
Sort by name/extension
[ ] Inverse long/short - если в панели отображаются длинные имена,
сортировать по коротким и наоборот.

10. Теперь Rename (F6) позволяет изменять регистр букв имени файла.

11. Options - Setup - Long filename support
[X] Short names "as is" - для показа коротких имен в том регистре, в
 котором они и есть.
