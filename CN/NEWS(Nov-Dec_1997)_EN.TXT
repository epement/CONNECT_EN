* Resizable columns in File Manager panels. Drag the panel dividers with
  the mouse, or switch to the Resize/move state (Ctrl+F5) and press
  Gray Plus/Gray Minus to expand/shrink panel columns. Names are displayed
  as long names when the column width is less or greater than 12.

* Mouse cursor masks are optionally supported
  (Options|Mouse|Disable cursor masks). These are used primarily to
  indicate drag/resize areas on window frames etc. The mouse cursor
  turns into a kind of arrow when it is moved over a hot spot, and
  is reset to its original state every time a key is pressed.

* Menus optionally track mouse (Options|Mouse|Track mouse in menus).
  When this option is checked, all program menus (including popups)
  handle mouse movements in a way similar to Win95 menus. However,
  you should click the button to open nested submenus like, for instance,
  File Manager popup Miscellaneous submenu item.

+ A new File Manager File compare option added: Prompt on compare.
  Checking this option causes the Compare options prompt dialog box
  to appear when Ctrl+C is pressed in a File Manager panel.

* A click on an Editor/Viewer row:column indicator opens the
  Go to line number dialog box (Alt+G). A click on a current character
  indicator opens an ASCII table (Alt+T).

+ New Editor options added: Wrap lines on Load, Unwrap lines on Save.
  They are useful for text files that don't contain CR/LF characters
  at the end of each line. Text lines are wrapped to Max line width
  and saved without CR/LF (CR/LF is written to mark a paragraph if a
  line is empty).
  Pressing Ctrl+O+] toggles the Wrap lines on Load option and reloads
  the text. Pressing Ctrl+O+[ toggles the Unwrap lines on Save option.

+ A new Edit menu item added: Save Clipboard data on exit.
  Checking this option causes CONNECT not to erase CNxxxxxx.@@@
  and CNxxxxxx.### Clipboard files on exit and re-use them later on
  the next session. Beware of their growing too large ! Uncheck this
  option every now and then to allow CONNECT to delete those files.

+ Scroll bars optionally added to File Manager panels
  (Alt+K - Panel settings dialog). Use Colors dialog to paint them,
  because those new palette entries most probably contain invalid
  zero values.

+ A new option added to the Drive list dialog (the one that opens
  when you press '.' in the Drive letter dialog box (Alt+F1/F2):
  [X] Open Drive list instead of Drive letter.
